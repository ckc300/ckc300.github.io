# 劉雨軒的Pathways動力中心

本網站是由 **劉雨軒(Yu-Xian Liu)** 建立的Toastmasters中文Pathways學習與資源輔助平台，  
提供清楚的學習路徑解析、專案資源整理，以及支援工具。

🌐 官方網站  
https://ckc300.github.io/

---

##這個網站提供什麼？

- Toastmasters Pathways六大現行路徑說明
- 已停用(Legacy)路徑的歷史與對照資料
- 專案計畫輔助內容
- 演講與例會用計時器
- 實務導向的學習整理(非官方教材)

本網站內容以 **中文使用者(國語/華語Toastmasters社群)** 為主要對象。

---

## 技術說明

- 純靜態網站(HTML / CSS / JavaScript)
- 使用GitHub Pages部署
- HTTPS由GitHub提供(自動TLS憑證)
- 適合桌機與行動裝置瀏覽
